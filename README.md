- Typescript

- 组件库 @antmjs/vantui

- 状态管理 mobx

### 开发

1. 安装依赖

```
yarn
```

2. 启动

```bash
# 开发环境
yarn dev:weapp

# 生产环境
yarn pro:weapp
```
