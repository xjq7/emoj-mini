## 小夏的表情包

### 技术栈

- Taro 3

- Typescript

- 组件库 @antmjs/vantui

- 状态管理 mobx

### 功能

- 浏览表情、表情包

- 收藏、点赞、下载、转发

### 开发

1. 安装依赖

```
yarn
```

2. 启动

```bash
# 开发环境
yarn dev:weapp

# 生产环境
yarn pro:weapp
```
